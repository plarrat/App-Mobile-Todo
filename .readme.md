# TD : App Mobile Todo

## Description

Application de base d'une todo List en React Native

## Installation

Récupérer le dépôt

`git clone https://github.com/plarrat/App-Mobile-Todo.git`

Se positionner dans le répertoire

`cd ./App-Mobile-Todo`

Installer les dépendances

`npm install`

Lancer l'application

`npm start`
